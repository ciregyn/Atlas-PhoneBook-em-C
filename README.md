# 📞 PhoneBook CRUD em C

Sistema de gerenciamento de contatos desenvolvido em linguagem C.

O projeto foi criado com foco em aprendizado de:

- Structs
- Modularização
- Manipulação de arquivos binários
- CRUD completo
- Organização de projetos em C
- Trabalho em equipe utilizando GitHub

---

# 🚀 Funcionalidades

✅ Cadastrar contatos  
✅ Listar contatos  
✅ Buscar contatos  
✅ Editar contatos  
✅ Remover contatos  
✅ Validação de CPF  
✅ Validação de Email  
✅ Validação de Telefone  
✅ Persistência em arquivo binário (`PhoneBook.dat`)

---

# 📂 Estrutura do Projeto

```txt
PhoneBook/
│
├── include/
│   ├── arquivo.h
│   ├── contato.h
│   ├── operacoes.h
│   └── validacao.h
│
├── src/
│   ├── arquivo.c
│   ├── cadastrar.c
│   ├── consultar.c
│   ├── editar.c
│   ├── remover.c
│   ├── validacao.c
│   └── main.c
│
└── PhoneBook.dat
```

---

# 🛠 Tecnologias Utilizadas

- Linguagem C
- GCC (MinGW)
- Git
- GitHub

---

# ▶️ Como Compilar

```bash
gcc src/*.c -Iinclude -o PhoneBook.exe
```

---

# ▶️ Como Executar

```bash
./PhoneBook.exe
```

No Windows PowerShell:

```powershell
.\PhoneBook.exe
```
